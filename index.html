Welcome to algorhythm.cc

This website should work fine in all current browsers, including Lynx, w3m, and other text-based browsers.

Hello, world!
This page started out as a collection of my old bookmarks/links to useful stuff. I created it as a backup, in case I were to lose my browser bookmarks some time.
I host some scripts and tools here, for "safekeeping", in case the authors' own sites were to disappear (which happens way too often).
If you find something useful -- great! If you don't find anything useful -- too bad, but this page wasn't meant to please everyone. Just me. :)

If you...
  have suggestions on links that should be here (maybe your own project?),
  have corrections to errors/typos or outdated scripts/tools,
  don't want your scripts/tools hosted here,
then feel free to send me a message at i~at~algorhythm~dot~cc, or /msg ikaroz on LinkNet/EFNet.

FTP Servers:
  * glftpd (C, *NIX): www.glftpd.eu, github.com/glftpd
    The best and most configurable FTP server imho. Active development.
  * LundFTPD (C, *NIX): www.lundman.net/wiki/index.php/LundFTPD
    Single process, non-blocking FTPD with internal user database.
  * cuftpd (Java, cross-platform): bitbucket.org/jevring/cuftpd
    Doesn't look like there's been any activity since ~2012.
  * ebftpd (C++, *NIX): github.com/jawr/ebftpd, github.com/glftpd/ebftpd, github.com/bioboy/ebftpd
    No active development.
  * DrFTPD (Java, cross-platform): github.com/drftpd-ng/drftpd3, drftpd.org/index.php/Main_Page
    Good idea, but should've been written in a different language. No active development for years.
